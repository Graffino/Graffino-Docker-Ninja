# Ninja

Webpack flow we use at Graffino for building websites using Handlebars, Stylus and ES6 Modules.

## Getting Started

```Bash
# Clone Project
git clone https://github.com/graffino/ninja.git
cd ninja

# Initialize New Git Workspace
rm -rf .git
git init

# Install Dependencies
yarn install

# Start Dev Server
yarn run dev
```
